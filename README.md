BudgetBites

Dette er et skoleprojekt, hvor målet er at skabe en brugervenlig og bæredygtig webshop til billige madvarer.

Funktioner
- Oversigt over alle produkter hentet fra et API.
- Filtrering baseret på kategorier (f.eks. Frugt og grønt).
- Produktdetaljevisning med fokus på enkelt design.

Teknologier
- HTML & CSS (med fokus på minimalistisk design).
- JavaScript til dynamisk indlæsning af data.
- JSONtil datastrukturering.

Bæredygtighed
Sitet er optimeret til at fylde så lidt som muligt for at mindske dataoverførsel og strømforbrug. Vi bruger ingen tunge biblioteker eller unødvendig grafik.

Hvordan kører man projektet?
1. Clone eller download mappen.
2. Åbn `index.html` via en Live Server i VS Code.
